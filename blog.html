Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM.
1. Selecting by ID, where document.getElementById("myElement") is used.
2. Multiple elements can be selected using Class, using document.getElementsByClassName("myClass").
3. For a specific Tag Name, document.getElementsByTagName("p").
4. Using CSS selectors, document.querySelector(".myClass")
Through all these methods, it is possible to control various elements of a webpage with JavaScript.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is that innerHTML can return and set content, including the full HTML code, so it supports HTML tags. innerText returns only visible text. On the other hand, textContent returns the full text, even retaining hidden text. Simply put, innerHTML allows you to add text along with HTML, innerText only takes the visible text, and textContent displays all the text, but does not process the HTML.

Question-3: What is event delegation in the DOM?

Event Delegation is a smart technique where instead of adding separate event listeners to each small element, an event is placed on their parent element. When a click or other event occurs on a child element, that event is caught by its parent. This eliminates the need to add multiple event listeners, making the code simpler and performing better. For example, if you have a long list, instead of placing separate events on each li, you can just place one event on the entire ul.

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event occurs on an element, it propagates to its parent, grandparent and up to the document. For example, when a button is clicked, the button first responds, then its parent div, etc. This is very useful for event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating, adding, and deleting new elements with JavaScript is very easy. First, a new element is created with createElement, then it is added to the DOM with appendChild. To delete an element, removeChild or remove directly is used. For example, a new div can be created, added to the body or any other parent, and then easily deleted.
